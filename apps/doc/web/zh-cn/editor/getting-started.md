
# 快速开始

## 打开编辑器
- 打开在线版本： [https://gavinyork.github.io/zephyr3d](https://gavinyork.github.io/zephyr3d)  
> 系统要求
> - IndexedDB 支持
> - WebGL/WebGPU 支持
> - ES6 模块支持

### 创建或打开项目

进入主页后可选择：  
- **Create Project** ：创建项目，项目将存储在 IndexedDB 中；  
- **Open Project** ：打开IndexedDB中已创建的项目；  
- **Open Remote Project** ：通过URL以只读方式打开在线的远程项目；  
- **Import Project** ：导入本地硬盘上的项目到IndexedDB；


<img src="https://cdn.zephyr3d.org/doc/assets/images/editor-main.png" width="1024"/>

**项目功能**
打开项目后，您可以：
- 管理项目资产
- 编辑场景内容
- 编写和管理脚本
- 配置项目设置
- 预览和发布项目
