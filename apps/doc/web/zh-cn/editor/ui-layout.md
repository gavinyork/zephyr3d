
# 编辑器界面

**界面组成**（从上到下）：
- **主菜单**：顶部菜单栏，包含所有功能入口
- **工具栏**：快捷操作按钮，提供常用工具
- **工作区**：主要的编辑操作区域
- **状态栏**（中央）：状态信息显示

工作区主要分为四大区域：

| 区域 | 功能说明 |
|------|-----------|
| **Scene Tree**（左） | 显示场景层级结构 |
| **Viewport**（中） | 3D 视图与编辑操作 |
| **Inspector Panel**（右） | 属性与组件参数编辑 |
| **Content Browser / Log View**（下） | 资源文件管理与日志查看 |

快捷键说明：
- `Ctrl + S` 保存场景；  
- 鼠标中键 平移，右键 旋转视角，滚轮或`Ctrl + 中键`缩放视角。
- `T` 切换到移动模式
- `R` 切换到旋转模式
- `S` 切换到缩放模式
- `ESC` 切换为选择模式
- `F` 视角定位到选中对象
