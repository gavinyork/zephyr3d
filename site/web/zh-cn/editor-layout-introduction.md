
# 用户界面

## 界面布局

Zephyr3d Editor 采用经典的多面板布局：

[图片占位：主界面完整截图]

**界面组成**（从上到下）：
- **主菜单**：顶部菜单栏，包含所有功能入口
- **工具栏**：快捷操作按钮，提供常用工具
- **场景树**（左侧）：显示当前场景的节点层次结构
- **属性编辑器**（右侧）：编辑选中节点的详细属性
- **底部面板**：资产视图和日志信息
- **场景视口**（中央）：主工作区域，实时渲染场景并支持交互编辑

## 视口交互

场景视口是编辑器的核心工作区域，支持：
- 实时场景渲染
- 鼠标和键盘交互
- 物体的可视化编辑
- 摄像机控制和导航
